Naïve Bayes Classifier

Naïve Bayes is a simple yet powerful classification algorithm based on Bayes' Theorem. It assumes that features are independent given the class label.

from river.naive_bayes import GaussianNB
from river.metrics import Accuracy

nb_model = GaussianNB()
metric = Accuracy()

for x, y in data_stream:
    y_pred = nb_model.predict_one(x)
    metric.update(y, y_pred)
    nb_model.learn_one(x, y)

Adaptive Random Forest Classifier

Adaptive Random Forest is an advanced ensemble learning method that adapts to evolving data streams by dynamically adjusting tree structures.

  • Maintains multiple decision trees for robustness.
  • Adapts to concept drift in real-time data streams.
  • Balances accuracy and computational efficiency.
from river.ensemble import AdaptiveRandomForestClassifier

arf = AdaptiveRandomForestClassifier()

for x, y in data_stream:
    arf.learn_one(x, y)
    y_pred = arf.predict_one(x)

Live Model Training and Accuracy Tracking

Tracking model accuracy during training helps identify performance trends and detect potential issues early.

from river.metrics import Accuracy

accuracy_metric = Accuracy()

for x, y in data_stream:
    y_pred = arf.predict_one(x)
    accuracy_metric.update(y, y_pred)
    print(f'Current Accuracy: {accuracy_metric.get()}')
